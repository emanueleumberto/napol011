Documnetazione API

Products API

  • GET: URL/api/products/ID (Get product by ID)
  • GET: URL/api/products (Get All products)
  • POST: URL/api/products (Create product)
  • PUT: URL/api/products/ID (Update product by ID)
  • DELETE: URL/api/products/ID (Delete product by ID)

Users API

  • GET: URL/api/users/ID (Get user by ID)
  • GET: URL/api/users (Get All users)
  • POST: URL/api/users (Create user)
  • DELETE: URL/api/users/ID (Delete user by ID)

Orders API

  • GET: URL/api/orders/ID (Get order by ID)
  • GET: URL/api/orders/user/ID (Get Orders by userID)
  • POST: URL/api/orders (Create order)
  • PUT: URL/api/orders/ID (Cancel order by ID)
